Automation [RPA]

Esta automação, desenvolvida em Python, lê arquivos Excel, processa os dados para extrair informações cruciais sobre vendas. O objetivo é gerar um relatório que é apresentado em um documento PDF. Este relatório é então salvo na área de trabalho do usuário, dentro de uma pasta específica.

Previsão do dólar [Machine Learning]

Em resumo, treinado com dados históricos, o código implementa um aplicativo para prever o valor do dólar. Ele utiliza a biblioteca Tkinter para a interface gráfica e a biblioteca Scikit-learn para o modelo de aprendizado de máquina. É interessante notar que este aplicativo pode ser aplicado em várias áreas, como previsão de aumento de custos, vendas e estoque.

SaaS [Software as a Service]

É um modelo que hospeda e gerencia softwares remotamente, permitindo acesso via internet por meio de assinaturas. Isso reduz custos e facilita a implementação, com provedores responsáveis pela manutenção e segurança.
